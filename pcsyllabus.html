Course Syllabus

Prompt Engineering

Prepared By
Dr. Nitin Mishra
Professor, Department of Computer Science & Engineering
Parul University, Vadodara

Course Information

Course Title Prompt Engineering
Programme B.Tech / BCA / B.Sc. (First Year)
Credits 3-0-2
Teaching Hours 45 Hours Theory + 30 Hours Practical
Course Type Skill Enhancement / Open Elective

Course Objectives

  • Understand the fundamentals of Artificial Intelligence and Generative AI.
  • Learn Prompt Engineering principles and techniques.
  • Develop effective prompts for various AI applications.
  • Apply AI tools for academics, programming, research and business.
  • Understand ethical and responsible use of Generative AI.

Course Outcomes (COs)

CO Description
CO1 Explain the concepts of Artificial Intelligence, Generative AI and Prompt Engineering.
CO2 Design structured prompts using appropriate prompting techniques.
CO3 Apply prompt engineering for education, programming, business and research.
CO4 Evaluate AI-generated responses and improve prompt quality.
CO5 Demonstrate ethical, secure and responsible use of Generative AI tools.

UNIT I
Introduction to Artificial Intelligence and Prompt Engineering (9 Hours)

Topics

  • Introduction to Artificial Intelligence
  • Machine Learning, Deep Learning and Generative AI
  • Large Language Models (LLMs)
  • History and Evolution of Chatbots
  • Introduction to Prompt Engineering
  • Components of a Prompt
  • Popular AI Platforms
  • Applications of Generative AI
  • Advantages and Limitations of AI

Laboratory

  • Explore ChatGPT, Gemini, Claude and Microsoft Copilot.
  • Compare AI-generated responses.

UNIT II
Fundamentals of Prompt Design (9 Hours)

Topics

  • Prompt Structure
  • Instruction Prompting
  • Role Prompting
  • Context Prompting
  • Persona Prompting
  • Zero-shot Prompting
  • One-shot Prompting
  • Few-shot Prompting
  • Prompt Refinement
  • Formatting AI Outputs

Laboratory

  • Create prompts for emails.
  • Generate summaries.
  • Create translations.
  • Generate reports.
  • Create question-answer prompts.

UNIT III
Advanced Prompt Engineering Techniques (9 Hours)

Topics

  • Chain-of-Thought Prompting
  • Prompt Chaining
  • Tree of Thoughts (Introduction)
  • Multi-turn Conversations
  • Prompt Templates
  • Structured Outputs (JSON, Tables, Markdown)
  • AI Response Evaluation
  • Prompt Optimization
  • Debugging AI Outputs

Laboratory

  • Create multi-step prompts.
  • Generate structured reports.
  • Create AI workflow prompts.

UNIT IV
Applications of Prompt Engineering (9 Hours)

Topics

  • Prompt Engineering for Programming
  • Prompt Engineering in Education
  • AI for Research Writing
  • Business Applications
  • Healthcare Applications
  • Image Generation Prompting
  • Presentation Generation
  • Code Generation
  • Data Analysis using AI

Laboratory

  • Generate Python programs.
  • Create HTML webpages.
  • Generate PowerPoint presentations.
  • Create SQL queries.
  • Generate research abstracts.
  • Create AI image prompts.

UNIT V
Responsible AI, Ethics and Future Trends (9 Hours)

Topics

  • Responsible AI
  • AI Ethics
  • Bias in AI Models
  • Hallucinations in LLMs
  • Prompt Injection Attacks
  • Privacy and Security
  • Copyright and Intellectual Property
  • AI Governance
  • Emerging AI Technologies
  • Future of Prompt Engineering

Laboratory

  • Identify hallucinations.
  • Improve AI reliability.
  • Ethical AI case studies.
  • Secure prompting exercises.

Suggested Laboratory Experiments

  1. Introduction to Generative AI tools.
  2. Prompt writing for text generation.
  3. Prompt engineering for summarization.
  4. Email and letter generation.
  5. Code generation using AI.
  6. Debugging programs using AI.
  7. HTML webpage generation.
  8. Research paper abstract generation.
  9. Presentation generation.
  10. Image prompt design.
  11. AI-assisted data analysis.
  12. Comparison of outputs from multiple AI platforms.
