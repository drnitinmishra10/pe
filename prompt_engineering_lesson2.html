Prompt Engineering Course — Lesson 2: Core Prompting Techniques

This lesson introduces practical prompting techniques used to improve the quality, consistency, and usefulness of AI responses.

1. Zero-Shot Prompting

Ask the AI to perform a task without providing examples.

Summarize the following article in 100 words.

2. One-Shot Prompting

Provide one example before asking the real question.

Input: Happy
Output: Sad

Input: Strong
Output:

3. Few-Shot Prompting

Provide several examples to teach the desired pattern.

Question: 2 + 2
Answer: 4

Question: 5 + 7
Answer: 12

Question: 9 + 8
Answer:

4. Chain-of-Thought Prompting

A store gives a 20% discount on a ₹500 item.
What is the final price? Show your calculations.

5. Step-by-Step Prompting

Step 1: Design the database.
Step 2: Create REST APIs.
Step 3: Build authentication.
Step 4: Create the frontend.
Step 5: Deploy the application.

6. Instruction Prompting

Explain Python for beginners in under 300 words.
Include:
- Variables
- Functions
- Loops

Do not include classes.

7. Persona Prompting

  • You are a senior software architect.
  • You are an experienced doctor.
  • You are a marketing consultant.

8. Delimiter Prompting

Context:
"""
Paste article here.
"""

Task:
Summarize the article.

Output:
Bullet points only.

9. Output Formatting

  • Markdown
  • JSON
  • Table
  • Numbered list
  • Code only

10. Prompt Chaining

  1. Brainstorm ideas
  2. Select the best one
  3. Create an outline
  4. Write the draft
  5. Edit
  6. Create SEO title
  7. Generate social posts

Combining Techniques

Role:
You are a senior technical writer.

Task:
Explain Docker.

Context:
Audience: First-year computer science students.

Constraints:
- Under 500 words
- Include one analogy
- End with three key takeaways

Output:
Markdown

Common Pitfalls

  • Combining unrelated requests.
  • Leaving out key context.
  • Changing requirements mid-prompt.
  • Giving conflicting instructions.

Practice Exercises

  1. Write a zero-shot prompt to summarize a research paper.
  2. Create a few-shot prompt that converts informal writing into professional language.
  3. Design a persona prompt for a cybersecurity consultant.
  4. Create a prompt chain for building a mobile app.
  5. Combine role, context, constraints, and output format in one prompt.
Key Takeaways
  • Zero-shot: No examples.
  • One-shot: One example.
  • Few-shot: Multiple examples.
  • Chain-of-thought: Ask for reasoning.
  • Step-by-step: Break complex tasks down.
  • Persona: Define the AI's role.
  • Delimiters: Separate context clearly.
  • Output formatting: Specify structure.
  • Prompt chaining: Build complex workflows incrementally.
