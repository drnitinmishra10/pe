🌳 Relationship Between AI, ML, DL and Generative AI

Artificial Intelligence (AI) │ ├── Machine Learning (ML) │ ├── Deep Learning (DL) │ └── Generative AI
Technology Purpose
AI Make machines intelligent
ML Learn from data
DL Learn complex patterns
Generative AI Create new content

🤖 What is Machine Learning?

Definition:
Machine Learning is a subset of Artificial Intelligence that enables systems to learn from data without being explicitly programmed.

Examples

  • Email Spam Detection
  • Fraud Detection
  • Netflix Recommendations
  • Weather Prediction
  • Face Recognition

🧠 What is Deep Learning?

Deep Learning is a specialized branch of Machine Learning that uses Artificial Neural Networks inspired by the human brain.
Human Brain Neural Network
Neurons Artificial Neurons
Learning Training
Memory Model Parameters

🎨 What is Generative AI?

Generative AI creates completely new content such as text, images, audio, video and computer code.

Examples

  • ChatGPT
  • Claude
  • Gemini
  • DALL·E
  • GitHub Copilot

⚖️ AI vs ML vs DL vs GenAI

Feature AI ML DL GenAI
Learns From Data ❌ ✅ ✅ ✅
Uses Neural Networks ❌ Optional ✅ ✅
Creates Content ❌ ❌ Limited ✅
